##Weather Forecast
App built using Kotlin using the following technologies:
1- Kotlin
2- MVVM design pattern
3- Retrofit2
4- ViewBinding
5- Coroutine
6- Weather Api
7- LiveData
8- Retrofit 2
9- Dagger-Hilt
